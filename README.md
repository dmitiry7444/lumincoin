# lumincoin
App for accounting of household finances with authorization

## Get start

### Test userData
```sh
login: test@test.ru
password: 12345678Qq
```
### Backend launch

```sh
cd backendp
npm install
npm run start
```
### Frontend launch
```sh
cd frontend
npm install
npm run dev
```
